# 常用命令

## 1. [[python]转换文件编码格式](https://www.jianshu.com/p/d5030db5da0e)

**查看文件**

```bash
$ file test
test: UTF-8 Unicode text
```

**转换**

```bash
$ iconv -f utf8 -t gbk test -o test.gbk
```

**效果**

```bash
$ file test*
test:          UTF-8 Unicode text
test.gbk:     ISO-8859 text
```

## 3. Linux统计文本行数

　　- c 统计字节数。
　　- l 统计行数。

　　- w 统计字数。

eg:

```
$ wc - lcw file1 file2
4 33 file1
7 52 file2
11 11 85 total
```
